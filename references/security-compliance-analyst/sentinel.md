## Microsoft Sentinel

### SIEM + SOAR
- **Data connectors** — ingest logs from Azure, M365, third-party, on-prem
- **Analytics rules** — detect threats using scheduled queries, ML, Microsoft threat intelligence
- **Incidents** — correlated alerts with investigation tools, entity mapping
- **Automation rules + Playbooks** — SOAR via Logic Apps for automated response
- **Workbooks** — visualization dashboards for security monitoring
- **Hunting queries** — proactive threat hunting with KQL

### Key Data Sources
- Entra ID sign-in and audit logs
- Azure Activity logs
- NSG flow logs
- Defender for Cloud alerts
- Azure Firewall logs
- Key Vault diagnostic logs
- DNS Analytics
- Office 365 audit logs
