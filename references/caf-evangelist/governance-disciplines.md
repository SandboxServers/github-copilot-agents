## Governance Disciplines

| Discipline | What It Controls | Azure Tools |
|-----------|-----------------|-------------|
| **Cost Management** | Budgets, spending, chargeback | Cost Management, budgets, advisor |
| **Security Baseline** | Security controls, threat protection | Defender for Cloud, Sentinel, Secure Score |
| **Identity Baseline** | Auth, RBAC, privileged access | Entra ID, PIM, conditional access |
| **Resource Consistency** | Standards, tagging, naming, regions | Azure Policy, management groups, tags |
| **Deployment Acceleration** | IaC, CI/CD, configuration management | ARM/Bicep/Terraform, Azure DevOps, GitHub |
